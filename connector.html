Aster connector for Claude

Build it in Claude. Send the link. The Aster connector publishes a self-contained HTML artifact (a design, report, or dashboard) as a live, shareable web page, and hands back a public link plus a private admin link.

“This rocks.”

Server

Aster is a remote MCP server over Streamable HTTP. There is one universal endpoint for everyone:

https://share.asterlabs.app/mcp

No account, no API key, no OAuth. The endpoint is anonymous and stateless.

Add it to Claude

  1. Open Settings → Connectors.
  2. Click +, then Add custom connector.
  3. Paste https://share.asterlabs.app/mcp and add it.
  4. Enable it in a chat, then ask Claude to publish an artifact to Aster.

The tool: publish_to_aster

The server exposes a single write tool. It takes the HTML you ask Claude to publish and returns the links to share and manage it.

FieldTypeNotes
artifact_htmlstring (required)A complete, self-contained HTML document.
titlestring (optional)Shown on the share page. Defaults to "Untitled link".

It returns a public share URL (anyone with the link can view) and a private admin URL (manage or delete the page). It is annotated destructiveHint: true because it creates content; it reads no other data.

Try it

Example use cases:

  • "Publish that landing page to Aster and give me the link to send my cofounder."
  • "Publish this report as a web page titled 'Q3 Status' and give me the share link."
  • "Publish this pricing table as a page, then give me the admin link so I can take it down after the call."

You can also call the server directly to see the tool list:

curl -s https://share.asterlabs.app/mcp \
  -H 'content-type: application/json' \
  -d '{"jsonrpc":"2.0","id":1,"method":"tools/list"}'

Data handling

  • Accessed: only the HTML and optional title you explicitly ask to publish. No files, conversation, memory, or other connectors.
  • Stored: the HTML plus minimal metadata (random viewer/admin keys, title, timestamps, view counts) in Aster's database on Cloudflare. No video is uploaded through the connector.
  • Transport: HTTPS/TLS only.
  • Retention: pages auto-delete after 30 days; you can delete one immediately with its admin link.
  • No accounts, no PII collected, never sold or shared.

About sensitive content. Connector-published pages are stored in plaintext and are viewable by anyone with the link. They are private and unlisted, but not encrypted. For sensitive material, use the Aster Mac app's zero-knowledge encrypted sharing, where content is encrypted on your device and the key never reaches the server.
